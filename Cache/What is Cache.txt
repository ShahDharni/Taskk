Cache is an information technology for the temporary storage(caching) for an web documents, such as web pages,images and other types of Web multimedia, to reduce server lag.

Caching is one of those methods which a website implements to become faster. It is cost efficient and saves a CPU processing time.

Django comes with a robust cache system that lets you save dynamic pages so they don't have to be calculated for each request.

You can cache the output of the specific views, you can cache only the pieces that are difficult to produce, or you can cache your entire site.

Following are the options for caching : 
1. Database caching
2. File system caching
3. Local Memory caching
